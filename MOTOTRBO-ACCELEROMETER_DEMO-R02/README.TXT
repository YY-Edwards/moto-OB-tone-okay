Current GOB Accelerometer Demo Application Version – R02.00
 
GOB ACCELEROMETER REVISION HISTORY
---------------------------------------------------
R02.00
-        Accelerometer demo project designed to operate in the GOB’s 3rd party application space (address 0x80010000).  Previous version was designed to operate in the GOB’s bootloader space (address 0x80000000), which is not permitted for GOB-based application software.
-        Accelerometer demo project is designed for use with the Visual Studio-based Atmel Studio IDE.  Previous project version was designed for use with the Eclipse-based AVR32 Studio IDE, which is no longer actively supported by Atmel.

-        Initial version release
